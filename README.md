# 📊 Cosmetics Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Cosmetics Sales Analysis Dashboard built using Power BI.  
The dashboard provides insights into sales performance, product categories, brand comparison, discount impact, and customer preferences.

The goal of this project is to transform raw data into meaningful visualizations that support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing product categories
- Compare brand-wise performance
- Study the impact of discounts on sales
- Understand customer preferences using ratings and skin types

---

## 🛠️ Tools & Technologies Used
- Power BI – Data visualization and dashboard creation  
- Excel / CSV Dataset – Data source  
- Data Modeling – Relationships, measures, and KPIs  

---

## 📊 Dashboard Features

### 🔹 1. Overview Analysis
- Total Sales, Quantity Sold, Average Rating
- Sales trend over years
- Category-wise sales comparison
- Stock distribution across categories
- Sales vs Discount relationship

---

### 🔹 2. Brand Analysis
- Average price, reviews, and total products
- Sales performance by category
- Ratings across launch years
- Brand-wise rating comparison

---

### 🔹 3. Discount & Product Analysis
- Impact of discount ranges on sales
- Product distribution using treemap
- Price vs discount analysis
- Discount comparison across brands
- Organic vs non-organic product distribution

---

### 🔹 4. Customer & Skin Type Analysis
- Skin type preferences across categories
- Brand-wise total sales
- Sales distribution by skin type
- Price vs rating relationship

---

## 📈 Key Insights
- Lipstick is the highest-selling product category  
- Discounts significantly influence sales performance  
- Certain brands dominate the cosmetics market  
- Customer ratings help identify product quality  
- Skin type plays an important role in purchasing behavior
